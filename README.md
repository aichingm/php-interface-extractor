php-interface-extractor
=======================

Create a php interface from php class in one easy command.

Usage
-----

With arguments: 

    #php extract-interface.php path/to/MyClass.php MyClass path/to/IMyClass.php IMyClass
        
Interactive mode:

    #php extract-interface.php -i
